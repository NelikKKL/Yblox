# Yblox
Yblox это frontend Яндекс игр - с некоротыми фишками    
# Функции
Блокировщик рекламы     
(и его включение - выключение)     
Удобный и приятный интерфейс    
Не нужен браузер для работы
# Где скачать?
Можно скачать с релиза или собрать по следующей инструкцие
# Сборка проекта
1. Установите [Python 3.10+](https://www.python.org/downloads/).
2. Склонируйте репозиторий:
  
   git clone https://github.com/NelikKKL/Yblox.git
   cd Yblox

3. Установите необходимые библиотеки:

   pip install -r requirements.txt

4. Запустите приложение:

   python main.py

# Сборка в EXE

Для создания одного исполняемого файла используйте готовый скрипт сборки:

python build_exe.py

После завершения ваш файл будет находиться в папке `dist/Yblox.exe`.

# 📦 Зависимости

Проект использует следующие основные библиотеки:
- `customtkinter` - современный UI.
- `pywebview` - для работы встроенного браузера.
- `pillow` - обработка изображений (миниатюр).
- `requests` & `beautifulsoup4` - парсинг игр.
- `pythonnet` - интеграция с .NET для WebView2.

# 📋 Требования

- Операционная система: Windows 10 или 11.
- Установленный **WebView2 Runtime** (обычно поставляется вместе с Microsoft Edge).
